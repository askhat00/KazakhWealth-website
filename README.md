# 🛍️ KazakhWealth

KazakhWealth is a modern minimalist e-commerce web application built with Django.  
It provides essential online store features such as user authentication, product browsing, cart management, and search functionality.

---

## 🚀 Features

- 🔐 User registration and authentication
- 👤 User profile management
- 🛒 Shopping cart (add / remove / update items)
- 🔎 Product search functionality
- 📦 Product catalog
- 💬 Session-based user system
- 🎨 Clean and minimalist responsive UI

---

## 🛠️ Tech Stack

- **Backend:** Django, Python  
- **Frontend:** HTML, CSS, JavaScript  
- **UI Framework:** Tailwind CSS  
- **Database:** PostgreSQL  

---

## 🎯 Project Highlights

- Modular Django architecture (apps-based structure)
- Fast and simple navigation
- Dynamic pages
- Fully responsive design for mobile and desktop
- Focus on clean UI and user experience
